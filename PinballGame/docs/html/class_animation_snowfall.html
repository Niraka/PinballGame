Detailed Description

A snowfall animation for adding a little more life to an otherwise static background.

The texture in use by the animation is not stored on the object, and thus must be kept in memory elsewhere for as long as the animation is required.

The animation should be notified every time the size of the window changes in order to produce a correct snowflake distribution.
