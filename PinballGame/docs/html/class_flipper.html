Detailed Description

A Flipper is an extension of the GameObject class that adds the functionality for moving in an arc-like manner around a point in a direction depending on the current state.

The states of the Flipper represent how it will behave at any given time. The states are: Resting (Doing nothing whilst orientated at the rest angle), rising (Moving upwards towards the peak angle) and falling (Moving downwards towards the rest angle).
