Detailed Description

The FontManager is a singleton font resource management class that stores sf::Font objects in a convenient central location.

All font objects stored within the FontManager are assigned (by the programmer) unique std::string keys that can be used to retrieve the object for as long as it is in memory.

As an sf::font is a fairly low-memory resource, loading all fonts that you intend to use upon launch and keeping them for the lifetime of the program is an advisable strategy.
