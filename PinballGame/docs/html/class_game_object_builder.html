Detailed Description

The GameObjectBuilder is a factory class for the production of GameObjects.

The builder should be used to parse vectors of strings and return pointers to created GameObjects, and should not be expected to produce the correct GameObjects when invalid parameter sets are supplied.

The builder is also not a file reader. Parsing strings from a file should be handled externally.
