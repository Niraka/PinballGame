void MapSnowTunnels::build ( )
virtual

Builds the Snow Tunnels map. The map data is loaded in from a fixed file location ("maps/snow_tunnels.txt"). Note that the map demands the existence of the following objects: "flipperLeft", "flipperRight".

void MapSnowTunnels::collisionOccurred ( std::string  sName1,
std::string  sName2 
)
virtual

This function is called by the collision manager whenever a collision occurs between two Collidable objects that exist on the map. Note that the order in which the names are received for a collisionOccurred event are essentially random. Additionally, a Collidable inherits its name from a GameObject, allowing for safe calls to getGameObject(sName1) etc etc.

Parameters
sName1The name of the first Collidable.
sName2The name of the second Collidable.
