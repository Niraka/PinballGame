Detailed Description

The PlayerController is a configurations handler for any settings that the player can modify.

Calling the "toggle" functionality on a setting will switch the setting to the opposite of its current true or false value.

The settings stored in the controller can be edited at any time, but should generally be done so through the use of a user interface and at a time where it is natural the the player (such as on an options menu).

The controller contains no functionality for loading or saving a configuration file, and will return to hardcoded default values whenever the game is relaunched.
