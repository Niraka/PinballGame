Detailed Description

The TextureManager is a singleton texture resource management class that stores sf::Texture objects in a convenient central location.

All texture objects stored within the TextureManager are assigned (by the programmer) unique std::string keys that can be used to retrieve the object for as long as it is in memory.

Note that both storing large numbers of sf::Texture objects and loading textures in to memory constantly are wasteful behaviours.
