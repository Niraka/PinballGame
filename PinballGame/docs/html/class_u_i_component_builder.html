Detailed Description

A UIComponentBuilder provides a method for the creation of a set of user interface components (UIText / UIButton / UIImage / etc) via the interpretation of a vector of strings.

The type of component built depends on the parameters passed to the builder. The returned pointer, however, will always be a UIComponent.

UIComponent * UIComponentBuilder::build ( std::vector< std::string >  strings,
UIButtonListener *  bl 
)

Builds a user interface component (EG: UIText, UIButton, UIImage..) using the vector of parameters and stores it on the heap. Returns a pointer to the UIComponent base class of the component. This function is unstable when presented "bad" or missing parameters.

Parameters
stringsA vector of strings containing the parameters for building the component.
blA pointer to a button listener to give to any components that require one.
Returns
A pointer to the UIComponent base class of the component.
